## 1. 마크다운학습하기

각종 마크다운 문법(중제목은 ##)

- (-하고 스페이스)
- `#`은 대제목
- `##` 은 중제목
- `###`은 소제목
- `를 하면 회색 네모가 생긴다.

  - (tab누르면 이렇게 취소하고 싶으면 shift tab)


- ---- 하면 가로로 긴~~~ 줄이 생긴다. (*** 도 마찬가지)

---

- 강조

- ```
  *문장*
  _문장_
  **문장**
  __문장__
  ~~문장~~
  ```

- *문장*

- _문장_

- **문장**

- __문장__

- ~~문장~~

mkdir =>파일 생성 (git에서)

mv=>파일이동(mv 옮길 파일명 옮길 장소)

ls=> 파일확인

git init=>은 실행

git status =>은 상태확인

**git add . 하면 관리목록에 추가!!!**

**git commit 실제로 저장 또는 스냅샷**

**git push** 요 강조 부분만 나중에는 반복하게 된다.
